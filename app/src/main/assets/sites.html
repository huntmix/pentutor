Админ панель - удобная вещь для администрирования сайтов.Но часто её оставляют без защиты, или с паролями по умолчанию...
Злоумышленник может найти эту панель и получить контроль над сайтом!

Для поиска панели нам пригодится Admin Panel Finder Вводим URL и ждём)Если что то нашло, то заходите и попробуйте стандартные пароли, такие как admin/admin

Защита:

  • Ставить сложные пароли
  • Никому не сообщать его
  • Ограничить кол-во запросов

Для попытки сбрутить пароль нам понадобится SuperBruter, вписываем свой сайт, ждём, и если пароль будет найден нам об этом оповестят

Полное сканирование

Утилиты

  • RDS
  • WAS
  • XAttacker

RapidScan - универсальный, комплексный инструмент для сканирования сайтов!Используется для относительно быстрой оценки защиты сайта

Для установки заходим и прописываем следующие команды!
git clone https://github.com/skavngr/rapidscan
cd rapidscan

После установки пишем
python rapidscan.py http://вашсайт
Ждём результатов сканирования, и уже после найденных уязвимостей, эксплортируем их!

WAScan - утилита которая способна выведать почти всю информацию о сайте, найти и протестировать возможные sql ошибки и выдать в конце удобный скриншот!
git clone https://github.com/m4ll0k/WAScan.git wascan
cd wascan
pip install BeautifulSoup
Запускаем сканирование:
python wascan.py --url http://ваш.сайт --scan 5
XAttacker - инструмент который найдет cms, и проверит на ней уязвимости, такие как залитие шела или другое!
Установка
git clone https://github.com/Moham3dRiahi/XAttacker
apt install perl
Запуск:
nano list.txt
Вписываем нужные сайты; Ctrl+x; Y; Enter
perl XAttacker.pl
Говорим что у нас есть список и пишем путь(list.txt)
Готово!
